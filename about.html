Dovednosti

  • HTML 5

    Snažím se dodržovat sémantiku struktury HTML tagů pro SEO a Google vyhledávač. Používám správný typ elementu pro daný typ obsahu. Vyvaruji se nadměrného používání DIV elementů. Pro classy se snažím vycházet z BEM (Block element modifier) pojmenovávací konvence. Naučil jsem se používat picture element, pro různé druhy rozlišení, zařízení, druhů formátů a následných fallbacků.

  • Css 3

    Základní stylování dokumentu. Zaměřuji se na používání flexboxu a responzivního layoutu. Ve spolupráci se Sass je celkem jednoduché vytvořit celkovou podobu stránky. V poslední době jsem se zaměřil na pokročilejší funkce jako "animate","transform","transition".

  • Sass

    Sass jako takový nepoužívám, ale používám z něho vycházející jazyk SCSS. Neuvěřitelně efektivní věc. Nejen při psaní kódu, ale i při přípravě celkové struktury a přípravě a údržbě assetů pro další projekty. Používám tyto jeho funkce. "nesting", "@mixiny", "@placeholdery", cykly a proměnné. Diky funkci "@use" a "@forward" což jsou následovníci funkce "@import", která je nyní považovaná za zastaralou, si můžu připravit konkretní sadu kódu na různá místa a později je připojit do hlavního SCSS kódu. Vytvářím si vlastní @funkce, například pro přepočet hodnoty pro line-height z px na em. Díky tomu ušetřím čas když vytvářím breakpointy. Nepoužívám VSCode extension, ale přímo Sass terminal s Dart Sassem.

  • Jquery

    Prozatím pro mne novinka se kterou se nyní zabývám. Objevuji možnosti jakým způsobem mohu kontrolovat elementy na stránce. Tvorba funkcí, traverzování po DOM na stránce.

  • Git

    Umím inicializovat verzovací systém pro projekt. Následně commitovat a pushovat do úložistě. Vytvořil jsem si privátní úložiště na Githubu a zároveň jsem si zprovoznil Gitlab úložiště na svém domácím Proxmox serveru. Na obě úložiště odesílám změny v projektu. Zkouším vytvářet větve pro různé verze projektu. Ještě se potřebuji naučit jak používat Git v týmu.

  • IT Admin

    Operační systémy: Windows a Linux
    Kancelářské systémy: Office a Libre Office.
    Virtualizační platformy: Virtual Box a Proxmox.
    V případě Linuxu jsem schopen nainstalovat, spustit a udržovat webservrové aplikace - Apache,Nginx,php. Nastavit load balancer, reverzní proxy atd. Zabezpečit linux server pomocí SSH a přehodit ho na jiný než defaultní 22 port a k tomu upravit nastavení Firewallu. Mám také základní znalosti CCNA certifikace a ovládání IT network periferií, switche a routery.
